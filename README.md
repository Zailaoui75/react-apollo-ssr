# Minimal React Apollo SSR

Fromages listing and view detail

## Features

* React 16
* Babel 7
* Webpack 4
* React Router 5
* HMR of webpack

## Installation

* `git clone git@github.com:zyhou/react-apollo-ssr.git`
* cd react-apollo-ssr
* make install
* make start-server 
* visit `http://localhost:3000`
